#  ❌⭕ Tic Tac Toe - Classic Game

A simple and interactive **Tic Tac Toe Game** built using HTML, CSS, and JavaScript. Play against another player and enjoy a clean, responsive UI with real-time win detection.

## 🎮 Gameplay

* ❌ Player X vs ⭕ Player O
* 🧠 Take turns marking the grid
* 🏆 First to align 3 marks (row, column, diagonal) wins
* 🤝 Game ends in a draw if all cells are filled



## ✨ Features

* 🎯 Real-time win detection
* 🔄 Restart / Reset game option
* 💻 Clean and minimal UI
* 📱 Responsive design
* ⚡ Instant move updates

---

## 🕹 Controls

* 🖱 Click on any empty cell to place your mark

---

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript

---

## ⚙️ How It Works

* The board is represented using an array
* Players alternate turns between X and O
* JavaScript checks winning combinations after each move
* Displays winner or draw result dynamically
